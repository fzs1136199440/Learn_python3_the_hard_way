# Escape-Sequences

| escapes      | what it does                                                 |
| ------------ | ------------------------------------------------------------ |
| `\\`         | 反斜杠`\`,比如`print('\\')`,这个表达式的输出值为`\`.         |
| `\'`         | 单纯的单引号`'`,它不是字符串书写格式首尾里的那个单引号。     |
| `\"`         | 同`\'`.                                                      |
| `\a`         |                                                              |
| `\b`         |                                                              |
| `\f`         |                                                              |
| `\n`         | linefeed(LF)                                                 |
| `\N{name}`   | Character named name in the Unicode database (Unicode only). Unicode数据库中的字符名，其中name是它的名字，仅Unicode适用。 |
| `\r`         | Carriage return(CR)                                          |
| `\t`         |                                                              |
| `\uxxxx`     |                                                              |
| `\Uxxxxxxxx` |                                                              |
| `\v`         |                                                              |
| `\000`       |                                                              |
| `\xhh`       |                                                              |
